# To Do List App
This application was created for my promotion in React JS and Typescript and is version 0.0.1. 

<!--Установка и работа с проектом-->
## To install app
1. Clone app from GitHub

    ```git clone ```

2. Move to project
    
    ```cd to-do-list-app```

3. Install node module

    ```npm install``` or ```npm i```

4. To run app

    ```npm start```
